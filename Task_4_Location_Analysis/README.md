# Location-Based Restaurant Analysis

## Objective
To analyze the geographical distribution of restaurants and identify location-based patterns
using exploratory data analysis and visualizations.

## Dataset
The dataset contains restaurant information including city, latitude, longitude,
ratings, and price range.

## Analysis Performed
- Identified cities with the highest number of restaurants
- Analyzed average restaurant ratings by city
- Studied the relationship between price range and ratings
- Visualized restaurant distribution using latitude and longitude

## Key Insights
- New Delhi and Gurgaon have the highest concentration of restaurants
- Some cities show high average ratings despite fewer restaurants, indicating quality-focused markets
- Higher price range restaurants generally receive better ratings
- Restaurants are densely clustered in urban regions

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
